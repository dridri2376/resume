# RESUME #

## Project explanation ##
=> this webpage is a basic web-version of my resume, made with Typescript, scss, and HTML

## SETUP ##
=> To install required depedencies in order to run project
```
npm run install
```


### compile SASS ###
```
npm run stw
```

### compile TS ###
```
npm run tsw
```
### browser hot reload ###
=> you can use the " live server " extension from your code editor
